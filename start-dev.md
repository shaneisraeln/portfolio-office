# Quick Start Development

## Windows Users

### Terminal 1 - Start OS Dev Server:
```cmd
cd portfolio-temp-website\os
npm run dev
```

### Terminal 2 - Start Portfolio Dev Server:
```cmd
cd portfolio-temp-website
npm run dev
```

### Open Browser:
Navigate to: `http://localhost:8080/?dev`

---

## What You'll See

1. **Loading Screen**: SIBIOS boot sequence with your name
2. **3D Office**: Navigate around the 3D environment
3. **Monitor**: Click on the monitor to see your custom OS
4. **ShaneOS**: 
   - Boot animation
   - Desktop with icons (About, Projects, Skills, Contact, Terminal)
   - Double-click icons to open windows
   - Drag windows around
   - Interactive terminal with commands

## Terminal Commands

Try these in the Terminal window:
- `help` - Show all commands
- `about` - Your bio
- `skills` - Your technical skills
- `projects` - Your projects
- `contact` - Contact information
- `whoami` - Current user
- `date` - Current date/time
- `clear` - Clear terminal

## Tips

- Use `WASD` or arrow keys to move in the 3D environment
- Click and drag to look around
- The OS runs in the monitor screen
- All windows are draggable
- Press ESC to exit the monitor view
