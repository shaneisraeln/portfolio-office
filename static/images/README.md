# Visual Assets for Shane Israel N Portfolio

This directory contains visual assets that need to be replaced with Shane's custom branding.

## Files to Replace

### Favicon Files
These files create the browser tab icon and app icons:
- `favicon.ico` - Main favicon file (16x16, 32x32, 48x48 multi-resolution)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `android-chrome-192x192.png` - 192x192 Android icon
- `android-chrome-512x512.png` - 512x512 Android icon
- `apple-touch-icon.png` - 180x180 Apple touch icon

### Social Media Preview Images
These images appear when the portfolio is shared on social media:
- `preview-new.jpg` - Main preview image (recommended: 1200x630px for optimal social media display)
- `preview-image.jpg` - Alternative preview image
- `preview.jpg` - Legacy preview image

## How to Replace

1. **Create your custom favicon**: Use a tool like [Favicon Generator](https://realfavicongenerator.net/) to create all required sizes from a single image
2. **Create your preview image**: Design a 1200x630px image that represents your portfolio
3. **Replace the files**: Simply replace the existing files with your new ones, keeping the same filenames
4. **Test**: Clear your browser cache and test that the new icons appear correctly
